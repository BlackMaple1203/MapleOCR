## 快捷键

| 功能       | 快捷键             |说明|自定义|
| ---------- | ------------------ |-|-|
| 打开文件/文件夹   | `⌥ + O`       |适用于批量识别与文档识别界面| ✅|
| 截图识别界面 | `⌘ + 1`       |打开截图识别界面| ✅|
| 批量识别界面 | `⌘ + 2`       |打开批量识别界面| ✅|
| 文档识别界面 | `⌘ + 3`       |打开文档识别界面| ✅|
| 二维码识别界面 | `⌘ + 4`       |打开二维码识别界面| ✅|
| 设置界面   | `⌘ + ,`       |打开设置界面| ❌|
| 关于界面  | `⌘ + I`       |打开关于界面| ❌|
